URL de login: http://localhost:8080/api/v1/auth/login

- admin@eduimpact.test

- admin3@eduimpact.test

Mot de passe: ChangeMe123!

Notes:
- Ces comptes sont créés par le script de seed Prisma: prisma/seed.ts
- Pour relancer le seed manuellement: depuis le dossier backend
    npx ts-node ./prisma/seed.ts
- Le backend écoute sur le port 8080 (modifiable via PORT dans backend/.env)

 Exemple de requête (PowerShell):
   curl -X POST http://localhost:8080/api/v1/auth/login \
     -H "Content-Type: application/json" \
     -d '{"email":"admin@eduimpact.test","password":"ChangeMe123!"}'

  Bootstrap automatique:
  - Si la base ne contient aucun Admin, la première tentative de connexion
    avec ADMIN_EMAIL et ADMIN_PASSWORD crée automatiquement un compte Admin.
  - Variables utilisées: ADMIN_EMAIL et ADMIN_PASSWORD dans backend/.env
